### DaddyJoke allows you to get a random joke instantly via JS

##

### Examples

#### Get a random joke:

```javascript
const Joke = require("daddyjoke");
Joke.getJoke();
```

#### About DaddyJoke:

```javascript
const Joke = require("daddyjoke");
Joke.about();
```
